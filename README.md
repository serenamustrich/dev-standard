## 🚀 快速开始

### AI角色

| 角色 | 标识 | 职责 |
|------|------|------|
| CEO (主Agent) | 🤖 | 主导对话，调用工作流，协调Agent |
| 需求Agent | 📋 | 需求分析 |
| UI设计Agent | 🎨 | 界面设计 |
| 开发Agent | 💻 | 代码开发 |
| 质量Agent | 🔍 | 测试验证 |
| 部署Agent | 🚀 | 环境部署 |
| 知识管理Agent | 📚 | 文档整理 |

### CEO职责

1. **每次对话**都必须调用工作流
2. **协调**各岗位Agent密切合作
3. **监控**任务进度和质量
4. **汇报**用户执行结果

### 工作流10阶段

```mermaid
flowchart TD
    A[📋 需求创建] --> B[👥 AI全员评审]
    B --> C[🎨 UI设计] --> D[👥 UI评审]
    D --> E[💻 开发设计] --> F[👥 开发评审]
    F --> G[📝 代码提交] --> H[🔍 测试]
    H --> I[🚀 部署] --> J[📚 知识沉淀]
```
